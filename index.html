AB.

Me chamo Alexandre ballardin,


Sou um Desenvolvedor Front-End com especialidades em HTML, CSS, JavaScript e React, assim como design voltado para experiência do usuário.


Comecei na área da tecnologia quando ingressei no último ano do colégio, sempre fui um garoto aficcionado por jogos e computadores, mas nunca havia tentado procurar mais a fundo sobre o assunto. Logo após ingressar me apaixonei rapidamente e comecei a estudar muito sobre várias tecnologias, mas sempre voltado para o design e front-end das aplicações.

PROJETOS

Meus principais projetos abaixo

Sticky NotePad


Nesse projeto desenvolvi uma ferramenta de StickyNotes, aonde o usuário pode criar notas, com a cor desejada, editalas, excluí-las e principalmente posicioná-las como quiser pela tela, usando a função de Drag & Drop. Utilizei as tecnologias de HTML e CSS para o design interativo, e JavaScript para o funcionamento do sistema.

Pomodoro Timer


Este projeto se trata de um site baseado no metodo pomodoro de estudo, a aplicação permite o usuário utilizar temporizadores de foco, descanso curto e longo, além de atiivar ou desativar a música. Além disso, o site possui uma aba de tarefas onde é possível cadastrar, editar e apagar tarefas que ficam salvas na localStorage do navegador. Utilizei HTML, CSS e JavaScript, seguindo os passos de um curso da Alura.
